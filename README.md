# Hearing NetHack

Hearing NetHack をインストールしてプレイしたい人は、読んでください。

---

## Hearing NetHack とは

NetHack 3.6.7 に **音声読み上げ（TTS）** と **ブラウザビューア** を追加した Hearing Edition です。  
ゲームの画面内容をブラウザ上で表示しながら、テキストを自動で読み上げることができます。

---

## 動作環境

- Windows 10 / 11（64ビット）
- ブラウザ Chrome 推奨

---

## インストール方法

1. このページ上部の **`Setup インストーラ`** フォルダを開く
2. **`NetHack367_Setup_x64.exe`** をダウンロードしてダブルクリック
3. インストーラの指示に従ってインストール（デフォルトのまま進めて OK）

インストール先：`C:\Program Files\NetHack 3.6.7`

---

## 起動方法

1. スタートメニューまたはデスクトップの **NetHack 3.6.7** をダブルクリック
2. `NetHack.exe` が起動すると同時に、**ブラウザが自動で開き**、ゲーム画面が表示されます

> すでにブラウザでビューアを開いている場合は、自動では開きません。

---

## ブラウザビューアの使い方

| ボタン | 機能 |
|--------|------|
| **Sound ON / OFF** | 音声読み上げの ON / OFF を切り替え |
| **Read Again** | 現在の画面を最初から読み上げ |
| **Read Status** | ステータス行（HP・レベルなど）を読み上げ |
| **Home** | 画面表示をクリア |

### 辞書機能

- 英語の単語にマウスを重ねると日本語訳がポップアップ表示されます
- **入力欄**に英語・日本語を入力して **Save** を押すと辞書に登録できます
- **Export** で辞書を JSON ファイルとして保存、**Import** で読み込めます
- **Clear Dictionary** で辞書を全消去（確認ダイアログあり）

---

## アンインストール方法

コントロールパネル →「プログラムのアンインストール」→「NetHack 3.6.7」を選択してアンインストール
